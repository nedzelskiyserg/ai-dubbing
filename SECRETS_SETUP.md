# Настройка секретов для GitHub Actions

## Для автоматической сборки с диаризацией

Чтобы токен Hugging Face использовался при сборке в GitHub Actions, нужно добавить его как секрет:

### Шаги:

1. Перейдите в настройки репозитория: `Settings` → `Secrets and variables` → `Actions`

2. Нажмите `New repository secret`

3. Заполните:
   - **Name**: `HF_TOKEN`
   - **Secret**: ваш Hugging Face токен (начинается с `hf_...`)

4. Нажмите `Add secret`

### Готово! 

Теперь при каждой сборке в GitHub Actions токен будет автоматически использоваться из секрета, и он не будет виден в логах или коде.

---

## Локальная разработка

Для локальной разработки токен уже сохранен в файле `.env` (который не попадает в git).

Если нужно обновить токен локально:
1. Откройте файл `.env` в корне проекта
2. Обновите значение `HF_TOKEN=...`
3. Перезапустите приложение
