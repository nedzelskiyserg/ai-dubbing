# Требования для работы приложения

## Критически важные компоненты

### 1. Python Backend

**Что это:**
- Основной сервер приложения, который обрабатывает видео, транскрибирует, переводит и создает дубляж

**Что происходит, если не собран:**
- ✅ **Есть fallback**: Приложение попытается использовать исходный Python скрипт `api_server.py`
- ⚠️ **Требования для fallback**:
  - Должен быть установлен Python 3.10+ на системе пользователя
  - Должны быть установлены все зависимости из `requirements.txt`
  - Если Python не установлен, приложение **не запустится**

**Рекомендация:**
- Всегда собирать Python backend через PyInstaller для удобства пользователей
- Это избавляет от необходимости устанавливать Python вручную

### 2. FFmpeg

**Что это:**
- Инструмент для обработки видео и аудио
- Используется для:
  - Склейки аудио и видео при скачивании с YouTube
  - Создания финального дублированного видео
  - Сжатия и обработки аудио

**Что происходит, если не собран:**
- ✅ **Есть fallback**: Приложение ищет FFmpeg в системе:
  - В системном PATH
  - В стандартных путях установки (Program Files, /usr/bin, и т.д.)
  - Рядом с исполняемым файлом
- ❌ **Если FFmpeg не найден**:
  - Скачивание видео с YouTube **не будет работать** (ошибка при склейке)
  - Создание финального видео **не будет работать**
  - Пользователь увидит ошибку: "FFmpeg не найден"

**Рекомендация:**
- Всегда включать FFmpeg в сборку для гарантированной работы
- Или требовать от пользователя установить FFmpeg вручную

## Сценарии работы

### ✅ Идеальный сценарий (все собрано):
- Python backend упакован → работает без установки Python
- FFmpeg включен → работает без установки FFmpeg
- Приложение полностью автономно

### ⚠️ Частичный fallback:
- Python backend не собран, но Python установлен → работает через исходный скрипт
- FFmpeg не включен, но установлен в системе → работает через системный FFmpeg

### ❌ Критическая ошибка:
- Python backend не собран И Python не установлен → **приложение не запустится**
- FFmpeg не включен И не установлен в системе → **обработка видео не работает**

## Рекомендации для сборки

1. **Всегда собирать Python backend** - это критически важно для удобства пользователей
2. **Всегда включать FFmpeg** - это необходимо для работы основных функций
3. **Проверять наличие компонентов** перед упаковкой в Electron

## Сообщения об ошибках

Приложение покажет понятные сообщения, если:
- Python backend не запустился (требуется установка Python)
- FFmpeg не найден (требуется установка FFmpeg)

Пользователь будет знать, что нужно установить для работы приложения.
